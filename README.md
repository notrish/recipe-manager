# Recipe Manager

Консольный учебный проект для управления рецептами и списком покупок.

## Описание

Проект содержит классы:

- `Ingredient` - ингредиент рецепта;
- `Recipe` - рецепт блюда;
- `ShoppingList` - итоговый список покупок;
- `DietaryRecipe` - рецепт с диетической категорией.

## Установка

```bash
pip install -r requirements.txt
```

## Запуск тестов

```bash
pytest
```
